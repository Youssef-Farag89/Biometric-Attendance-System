# Biometric Attendance System

## Project Structure :
- `forms/` → UI files
- `headers/` → Header files (.h)
- `sources/` → Source files (.cpp)
- `img/` → Images for GUI
- `sfx/` → Sound assets
- `main.cpp` → Main entry point
- `resources.qrc` → Qt resources
- `BiometricAttendance.pro` → Qt project file

## Requirements : 
- Qt Creator (tested on version 6.x)
- C++ compiler compatible with Qt

## How to run
1. Open `BiometricAttendance.pro` in Qt Creator.
2. Build the project.
3. Run the application.

 ## Do not include
- Build folders
- Executables (.exe)
- Qt Creator settings (.user files)
